Windows beta setup

Install FlowBridge in a few simple steps.

FlowBridge has two parts: the desktop bridge app and the browser extension. Install both once, then use the Bridge button inside your AI chats.

1

Download the beta ZIP

Use the download link from your email, then extract the ZIP file.

2

Run FlowBridge setup

Open the setup file inside the extracted folder.

!

If Windows SmartScreen appears

Click More info, then click Run anyway. This beta is not code-signed yet.

3

Add the extension to Chrome

Open chrome://extensions, turn on Developer mode, click Load unpacked, then choose the extension folder.

4

Open an AI chat

Go to ChatGPT, Claude, Gemini, or Grok. Code blocks should show the FlowBridge button.

5

Test one command

Click the Bridge button, approve the command, then check that the terminal output returns correctly.
