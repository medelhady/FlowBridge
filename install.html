Windows beta setup

Install FlowBridge in a few simple steps.

FlowBridge has two parts: the VS Code terminal bridge and the browser extension. Install both once, then use the Bridge button inside your AI chats.

1

Download FlowBridge

Use the download button above or the link from your email, then extract the ZIP file.

2

Run FlowBridge setup

Open the extracted folder and double-click Install FlowBridge.bat. This installs the VS Code bridge and opens the Chrome extensions page.

!

If Windows shows a security warning

Click Run. Windows may show Unknown Publisher because this beta installer is not code-signed yet.

3

Add the extension to Chrome

Open chrome://extensions, turn on Developer mode, click Load unpacked, then choose the browser-extension folder.

4

Check VS Code

Open VS Code and make sure the bottom bar shows FlowBridge: On. The beta access lasts 14 days from first run.

5

Open an AI chat

Go to ChatGPT, Claude, Gemini, or Grok. Code blocks should show the FlowBridge button.

6

Test one command

Click the Bridge button, approve the command inside VS Code, then check that the terminal output returns correctly.
